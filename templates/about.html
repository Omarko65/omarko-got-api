What Is This?

OMARKO-GOT-API is a Game Of Thrones API that returns Game Of Thrones data.

I am a big fan of Game of Thrones and movies in general, for my ALX portfolio project i always knew my project was going to be movie related.

This project returns data in JSON format based on what is requested. You can check the DOCUMENTATION to get tips on how to use this application.

What Are The Features

Although the original project uses DJANGO and DJANGO REST Framework to serve the api, this one uses FLASK Framework to serve the api.
